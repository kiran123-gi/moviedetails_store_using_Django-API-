# Movie Data API (Django REST Framework)

This project is built using Django REST Framework to display movie data from the database on a web page.

## 🚀 Features
- Created REST API using Django REST Framework
- Stored movie data in database
- Fetched and displayed data on web page
- Basic serialization and API handling

## 🧠 What I Learned
- How to build APIs using Django REST Framework
- Serialization of data
- Fetching data from backend to frontend
- Understanding request and response cycle

## 🛠️ Tech Stack
- Python
- Django
- Django REST Framework
- HTML
- CSS

## 📌 Project Goal
To understand how backend APIs work and how data is transferred from database to frontend.

## 🔗 Future Improvements
- Add POST, PUT, DELETE APIs
- Connect with React frontend
- Add authentication
